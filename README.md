# tabela-de-classificacao
Tabela para gerenciamento de pontos de um jogo 1 vs 1, no qual é possível:
- adicionar o nome dos jogadores;
- selecionar, dentre os jogadores adicionados, os 2 jogadores da rodada;
- classificar o vencedor da rodada ou se houve empate;
- calcular automaticamente os pontos de cada jogador;
    O cálculo dos pontos será da seguinte forma:
    - 1 vitória => ganha 3 pontos
    - 1 derrota => perde 1 ponto
    - 1 empate => ganha 2 pontos
- Remover jogador da tabela de classificação;
- Zerar a pontuação da tabela.
